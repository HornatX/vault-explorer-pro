## 简体中文

### 核心功能

#### 1. 文件夹快捷图标
在任意文件夹旁添加快捷图标，一键直达常用笔记，支持自定义图标和颜色。

#### 2. 文件内联附属
将关联文件挂载在主文件名后显示，原文件自动隐藏，实现无缝嵌套效果。

#### 3. 文件夹锁定
锁定指定文件夹，防止误触展开，点击时显示提示，保护文件结构稳定。

#### 4. 文件显示限制
为文件夹设置显示数量上限，超出部分自动折叠，支持保留前N个或后N个。

#### 5. 自动源码模式
打开指定文件时自动切换到源码编辑模式，无需手动切换，提升编辑效率。

#### 6. 自动收起属性
打开Markdown文件时自动折叠顶部属性面板，让内容区域更宽敞。

#### 7. 隐藏文件夹
完全隐藏指定文件夹，简化侧边栏视图，支持命令面板快速切换显示状态。

***

## 用法

### 文件夹快捷图标
1. 右键点击任意文件夹 → 选择「设置快捷文件」。
2. 在弹窗中点击「+ 添加文件夹内部快捷文件」。
3. 选择目标文件，可自定义图标和颜色。
4. 快捷图标将显示在折叠的文件夹旁，点击即可打开。

### 文件内联附属
1. 右键点击任意文件 → 选择「设置内联文件」。
2. 选择同目录下的关联文件。
3. 关联文件将显示在主文件名后，原文件自动隐藏。

### 文件夹锁定
1. 右键点击文件夹 → 选择「锁定文件夹」。
2. 锁定后点击文件夹会提示「文件夹已锁定」。
3. 需要展开时右键选择「解锁文件夹」。

### 文件显示限制
1. 打开设置 → 找到「文件夹文件显示个数限制」。
2. 点击「添加文件显示个数」，输入文件夹路径和数量。
3. 可选择保留「前N个」或「后N个」文件。
4. 超出数量的文件会自动折叠，点击边缘图标展开。

### 自动源码模式
1. 打开设置 → 启用「自动源码模式」。
2. 在文本框中输入需要自动切换的文件名（每行一个）。
3. 打开对应文件时将自动进入源码编辑模式。

### 自动收起属性
1. 打开设置 → 启用「自动收起笔记属性面板」。
2. 打开任意Markdown文件时，顶部属性面板将自动折叠。

### 隐藏文件夹
1. 打开设置 → 找到「完全隐藏的文件夹」。
2. 手动输入路径或点击「🔍 浏览」按钮选择文件夹。
3. 使用命令面板（Ctrl/Cmd+P）搜索「切换隐藏文件夹」可快速切换显示状态。

#### 方法一：社区插件安装（推荐）

4. 将下载的三个文件放入该文件夹中。
5. 在 Obsidian **设置** > **社区插件** 中重新加载并开启该插件。

## English

**Vault Explorer Pro** — An all-in-one sidebar enhancement plugin for Obsidian featuring **folder shortcuts, inline file nesting, folder locking, dynamic file limits, automatic source mode, and hidden folders**.

***

### Features

#### 1. Folder Shortcut Icons
Add quick-access icons next to any folder for instant access to important notes, with custom icons and colors.

#### 2. Inline File Attachment
Mount related files inline after the main filename, original files auto-hidden for seamless nesting.

#### 3. Folder Locking
Lock specific folders to prevent accidental expansion, showing notifications on click to protect file structure.

#### 4. File Display Limits
Set display limits for folders, auto-collapse excess files, support keeping first N or last N files.

#### 5. Auto Source Mode
Automatically switch to source editing mode when opening specified files, no manual switching needed.

#### 6. Auto Collapse Properties
Automatically collapse the properties panel when opening Markdown files for more content space.

#### 7. Hidden Folders
Completely hide specified folders from sidebar, with command palette toggle for quick visibility control.

***

## Usage

### Folder Shortcut Icons
1. Right-click any folder → Select "Set Shortcut Files".
2. Click "+ Add internal shortcut file" in the popup.
3. Select target file, customize icon and color.
4. Shortcut icon appears next to collapsed folder, click to open.

### Inline File Attachment
1. Right-click any file → Select "Set Inline File".
2. Select related file from same directory.
3. Related file displays after main filename, original auto-hidden.

### Folder Locking
1. Right-click folder → Select "Lock Folder".
2. Clicking locked folder shows "Folder is locked" notification.
3. Right-click → "Unlock Folder" when expansion needed.

### File Display Limits
1. Open Settings → Find "Folder File Display Limit".
2. Click "Add File Display Limit", enter folder path and count.
3. Choose to keep "First N" or "Last N" files.
4. Excess files auto-collapse, click edge icon to expand.

### Auto Source Mode
1. Open Settings → Enable "Auto Source Mode".
2. Enter filenames in textarea (one per line).
3. Opening corresponding files auto-enters source editing mode.

### Auto Collapse Properties
1. Open Settings → Enable "Auto Collapse Properties Panel".
2. Properties panel auto-collapses when opening any Markdown file.

### Hidden Folders
1. Open Settings → Find "Completely Hidden Folders".
2. Manually enter path or click "🔍 Browse" to select folder.
3. Use Command Palette (Ctrl/Cmd+P) → "Toggle Hidden Folders" for quick visibility control.

***

### Installation

#### Method 1: Community Plugins (Recommended)

Once the plugin is reviewed and listed on the community marketplace:
1. Open Obsidian **Settings** > **Community plugins** > **Browse**.
2. Search for and select `Vault Explorer Pro`.
3. Click **Install** and then **Enable**.

#### Method 2: Manual Installation

1. Go to the [Releases](https://github.com/hornatx/vault-explorer-pro/releases) page to download the latest `main.js`, `manifest.json` and `styles.css` files.
2. Open your Obsidian vault folder on your computer.
3. Navigate to the `.obsidian/plugins/` directory and create a folder named `vault-explorer-pro`.
4. Place the downloaded files into this folder.
5. Reload and enable the plugin in Obsidian **Settings** > **Community plugins**.
